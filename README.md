## Local Setup

1. Install the dependencies and start the server.

```sh
git clone https://github.com/adarsharyan002/Vid.io.git

cd Vid.io
npm install
cd client
npm install
```


```

4. Start the development server from the Client folder.

```sh
npm start
```
